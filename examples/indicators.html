Loading Indicators

Different indicator styles shown during API requests. The server has a 400ms delay to make indicators visible.

1. Text indicator

A simple text message inside the element with class xh-indicator.

Loading users...

2. CSS spinner

A rotating border spinner next to the button.

Loading...

3. Pulsing dots

Animated dots as a loading indicator.

4. Skeleton placeholder

Skeleton lines that appear while data is loading.

5. Progress bar

An indeterminate progress bar at the top of the result area.

6. Overlay with spinner

A semi-transparent overlay with a spinner on top of existing content.

Click "Load Users" to see the overlay.

How indicators work

<!-- Point to any element as the indicator -->
<button xh-get="/api/data"
        xh-indicator="#my-spinner">Load</button>
<span id="my-spinner" class="xh-indicator">Loading...</span>

<!-- xhtmlx auto-injects this CSS: -->
<!-- .xh-indicator { opacity: 0; transition: opacity 200ms; } -->
<!-- .xh-request .xh-indicator,                                -->
<!-- .xh-request.xh-indicator { opacity: 1; }                  -->

<!-- During a request, the xh-request class is added -->
<!-- to the indicator element, making it visible.     -->
