xhtmlx Examples

Declarative HTML attributes for REST API driven UIs

  • Basic GET & Data Binding Load a user on page load, display fields with xh-text and xh-attr-*, loading indicator.
  • CRUD Application Full create, read, update, delete operations with external templates.
  • Nested Templates Users → posts → comments with nested API calls and $parent access.
  • Iteration (xh-each) Simple lists, nested iteration, $index usage.
  • Conditionals (xh-if / xh-unless) Show/hide elements based on data fields: admin badges, verified status.
  • Triggers click, load, keyup with delay (live search), every 5s (polling), revealed.
  • Loading Indicators Spinner and skeleton styles shown during requests.
  • Error Handling Per-status-code error templates, validation errors, generic fallback.
  • Error Boundaries Three levels of error handling: element-level, error boundaries, and global config.
